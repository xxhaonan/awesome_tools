## ⚒️ Try it out!
修改`reneme.py`中原始文件夹所在目录的地址
```console
directory = r"C:\Users\user\Desktop\origin"
```
修改`reneme.py`中要替换的旧文字
```console
old_string = "cornell"
```
修改`reneme.py`中要替换的新文字
```console
new_string = "clutter"
```
运行程序
```console
python rename.py
```
